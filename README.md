# osvr-device-descriptor-editor

> Browser-based editor for OSVR device descriptor JSON..

## Build instructions
You need `npm`, `grunt`, and `bower` installed globally. I think.

```bash
npm i && grunt
```

Output ends up in `dist`.

You can serve it locally with

```bash
grunt server
```
